Welcome!

This plugin came with the need to implement a custom autocomplete on a website. With it you can pick up information provided in json in a given url or json generate on your own site and passes it as a parameter.

Example



How to use (basic way)

* Dependencies: JQuery (preferably the latest version)

No seu arquivo html:

<div id="buscaTopo">
    <input type="text" name="term" placeholder="Find friends..." value="" />
</div>

Na tag script:

$( "#buscaTopo" ).autocomplete({
  //Soon documentation explaining how best to exploit the parameters
});

And do not forget to add the library files into your page:

<script src="js/autocomplete-custom.js" />
<link rel="stylesheet" href="css/autocomplete-custom.css" />
